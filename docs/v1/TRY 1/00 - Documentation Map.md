# 1. Purpose

This documentation set converts the Knowledge Factory discovery conversation into structured development documents for Knowledge Factory Version 1.

Computer-OS wording is intentionally avoided. Knowledge Factory is defined as a professional knowledge manufacturing platform and architecture, not a Windows, macOS, or Linux equivalent.
