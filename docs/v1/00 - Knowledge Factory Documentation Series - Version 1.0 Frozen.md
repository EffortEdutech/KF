# Knowledge Factory (KF)

## Documentation Series - Version 1.0 (Frozen)

This document defines the current Knowledge Factory Version 1 documentation set.

The source of truth for this index is the `docs/v1` folder. Each document is a standalone Markdown file and follows the hierarchical KF numbering scheme.

---

# 1. Documentation Philosophy

Knowledge Factory documentation is structured as an architecture series rather than a simple sequential book.

The numbering communicates each document's role:

- `KF 1_x` defines the foundation.
- `KF 2_x` defines the core platform.
- `KF 3_x` defines governance and knowledge products.
- `KF 4_x` defines developer and enterprise capabilities.
- `KF 5_x` defines future vision, ecosystem strategy, LADOS integration, deployment direction, and long-term roadmap.

This structure allows future documents to be added without renumbering existing work.
